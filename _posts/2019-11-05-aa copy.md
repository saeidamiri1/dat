## Variable name using loop object
If you need to create the variable name using the loop object, use `i`:
````
for i in range(4):
    exec(f'var_{i} = [range(i)]')
````

I personally prefer using the directory instead the variable. 
````
var={}
for i in range(4):
    var[f'var_{i}']=range(i)
````
